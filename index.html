Elevator pitch - design system

Colors

The project have derived it's colors from the picture of the woman in the elevator. The colors used are:

Main background color rgb(144 163 173)
Elevator background color rgb(146 157 110)
Heading/paragraph color (H2, H3) rgb(14 20 27)
Link color rgb(206, 201, 200)
Bordercolor rgb(181 181 178)

Fonts

As a start there is only one font that is used for the whole page:

  • Abel from Google Fonts.

Borders

Each "floor"/section of the page and every picture are styled to have rounded edges and a thin border.

border-radius: 10%;
border: 1px solid rgb(181 181 178);

Text styles

The text is styled as follows:

The h1 has a background-color of rgb(144 163 173) and color of white.

h1 {
padding: 20px;
color: white;
font-size: 28px;
text-align: center;
text-transform: uppercase;
letter-spacing: 0.3em;
}

This is the style of a h1 header

The other headers h2 and h3 has a background-color of rgb(146 157 110) and the color of rgb(14 20 27)

h3{
font-family: 'Abel', sans-serif;
background-color: rgb(146 157 110);
color: rgb(14 20 27);
font-size: 50px;

}
